Crie uma nova pasta no seu computador para o projeto e dentro dela crie os arquivos index.html e estilo.css; -ok

Estruture seu site usando tags semânticas: utilize <header> para o cabeçalho,
 <nav> para o menu de navegação das notícias (dentro do <nav> use listas <ul> ou <ol>), <main> para agrupar todo o conteúdo principal, <section> e <article> para separar e organizar cada notícia individual, e <footer> para o rodapé com informações de contato, endereço e redes sociais fictícias;-
 
- Criar: header dentro do body com o título "Portal de Notícias"-ok
- Criar: nav dentro do header com uma lista não ordenada (ul) contendo links para "Home", "Notícias", "Esportes", "Entretenimento" e "Contato"-ok
- Criar: main para o conteúdo principal-ok
- Dentro do main, criar duas sections, cada uma contendo um article com uma notícia fictícia-ok
- Cada article deve ter um título (h2), um parágrafo (p) com o conteúdo da notícia e uma data (time)-ok
- Criar: footer com informações de contato, endereço e links fictícios para redes sociais-quase acabando-ok

Em alguma das notícias, inclua um vídeo usando a tag <iframe>;-ok

Por fim, estilize seu site com o CSS, aplicando cores de fundo e das letras, tipos e tamanhos de fontes.
